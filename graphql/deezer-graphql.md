# Deezer GraphQL API

Deezer is a music streaming service. The API covers track search, artist and album lookups, playlist management, user library, editorial playlists, radio, podcast episode data, and music recommendations.

**Endpoint:** No native GraphQL endpoint — conceptual schema derived from Deezer REST API
**Documentation:** https://developers.deezer.com/api

- Reference: https://developers.deezer.com/api
- Schema: https://github.com/api-evangelist/deezer
